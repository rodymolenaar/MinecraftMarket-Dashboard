Global commands

Command variables

{name} - This variable will be replaced with the customers username.

{uuid} - This variable will be replaced with the customers UUID.

{transactionid} - This variable will be replaced with the transaction id.

{price} - This variable will be replaced with the overall price paid by the customer.

{currency} - This variable will be replaced with the currency set for this payment which is also the market currency.

{time} - This variable will be replaced with the time of purchase.

{date} - This variable will be replaced with the date of purchase.

{email} - This variable will be replaced with the customers email when applicable.

{ip} - This variable will be replaced with the customers ip that is recorded on checkout.

{itemid} - This variable will be replaced with the ID of the item this command is associated with.

{itemprice} - This variable will be replaced with the price of the item this command is associated with.

{itemname} - This variable will be replaced with the name of the item this command is associated with.
